## Backend API

> Interview Take Home Project for DreamzTech Solutions. Available API endpoints are `GET` and `POST` for the tax list. State list data also stored in MongoDB atlas and `GET` endpoint for this also available.

## Backend Set-up
- `npm i`
- `npm run serve`

## Built With
- Node JS
- Express JS
- MongoDB
