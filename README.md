# Interview Take Home Project

> It's a **MERN** stack project for [DreamzTech Solutions](https://dreamztech.com) coding round. Followed GitHub flow.


## Built With

- FrontEnd:
  - React JS
  - Redux
- Backend:
  - Node JS
  - Express JS
- Database:
  - MongoDB

## Getting Started

- `git clone https://github.com/RitobrotoMukherjee/dreamz-tech.git`
- `cd dreamz-tech`

## Backend Set-up
- `cd backend`
- `npm i`
- `npm run server`
- `cd ..`

### Frontend Set-up
- `cd frontend`
- `npm i`
- `npm run start`
- `cd ..`
